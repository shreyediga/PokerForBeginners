Betting Over Others

3 Betting

Linear 3Betting - This is where we 3bet for value and only bet if we have a better range than the Villian.
Polar 3Betting - This is where we have both bluffs and value hands. Here we 3bet the really good hands and call with the good enough hands.

We use linear when we have limited fold equity or we don't want to just call. We don't need to have a call range when using linear since we only really play good hands.

4 Betting

Altough you don't 4bet often you do sometimes if you have amazing hands like AA or AKs. When we 4bet we do not care if villian thinks we have a strong hand since we are just trying to grow the pot. As you can see, almost every single hand is folded here.

Sizing Bets

Sizing a bet is very important and if done incorrectly can lead to losing so much more than you should.
3Bet:
   Hero should raise to 3-3.1x and add an additional 1x for every caller and another additional 1x if out of position with x being the current raise size.
4Bet:
   Hero should raise to 2.1x or 2.3x if out of position and add an additional .75x for every caller with x being the current raise size.
